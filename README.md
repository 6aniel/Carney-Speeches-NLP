# NLP-Project-1
personal project in the works...

NLP Analysis of Mark Carney's Political Rhetoric

This project analyzes Mark Carney's speeches and public remarks using NLP methods in R. 
The initial dataset is scraped from the official Prime Minister of Canada website.

Current steps:
1. Collect speech URLs
2. Scrape speech text into CSV format
3. Clean and tokenize text
4. Analyze word frequency and themes
